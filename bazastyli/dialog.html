Dokumentacja komponentów dialogu

Poniżej znajduje się opis klas CSS i struktur HTML do budowania dialogów/modali w systemie CRM.

1. Struktura dialogu

Klasa Opis Użycie
.dialog-modal Główny kontener dialogu Wrapper dla całego modala
.dialog-header Nagłówek z tytułem i przyciskiem zamknięcia Zawiera .dialog-title i .dialog-close
.dialog-actions Pasek akcji pod nagłówkiem Zawiera .dialog-actions-left i .dialog-actions-right
.dialog-body Główna zawartość dialogu Formularz lub inna zawartość
.dialog-footer Stopka z przyciskami akcji Zamknij, Zapisz, etc.

2. Layout formularza

Klasa Opis
.form-grid Grid 2-kolumnowy dla formularza (responsive)
.form-column Pojedyncza kolumna formularza
.form-section Sekcja formularza (np. "Ogólne informacje")
.form-section-title Tytuł sekcji z zieloną kreską
.form-section-content Zawartość sekcji (pola formularza)

3. Pola formularza (label po lewej)

Klasa Opis
.form-row-horizontal Wiersz: label (180px) + input
.form-label-horizontal Label w kolorze niebieskim
.form-label-horizontal.required Label z gwiazdką (*) dla pól wymaganych
.form-input Pole tekstowe
.form-select-custom Pole select ze strzałką
.form-row-checkbox Wiersz z checkboxem
.form-row-range Wiersz z polami Od/Do
.form-input-clearable Wrapper dla inputa z przyciskiem czyszczenia (X)
.form-input-icon-wrapper Wrapper dla inputa z ikoną (np. kalendarz)
.form-column-narrow Kolumna z węższymi labelami (140px)

4. Przyciski

Klasa Opis Kolor
.btn-dialog-primary Główny przycisk akcji Zielony (#8ebf29)
.btn-dialog-secondary Przycisk anulowania Szary outline
.btn-dialog-info Przycisk informacyjny Niebieski (#0d6efd)

5. Przykład kodu - Sekcja formularza

<div class="form-section"> <h6 class="form-section-title">Nazwa sekcji</h6> <div class="form-section-content"> <!-- Pole tekstowe --> <div class="form-row-horizontal"> <label class="form-label-horizontal required">Nazwa pola</label> <input type="text" class="form-input"> </div> <!-- Pole select --> <div class="form-row-horizontal"> <label class="form-label-horizontal">Wybór</label> <select class="form-select-custom"> <option>Opcja 1</option> </select> </div> <!-- Checkbox --> <div class="form-row-checkbox"> <label class="form-label-horizontal">Zgoda</label> <div class="form-checkbox-wrapper"> <input type="checkbox" class="form-checkbox"> </div> </div> <!-- Pola Od/Do --> <div class="form-row-range"> <label class="form-label-horizontal">Zakres</label> <div class="form-range-inputs"> <span class="form-range-label">Od</span> <input type="text" class="form-range-input"> <span class="form-range-label">Do</span> <input type="text" class="form-range-input"> </div> </div> </div> </div>

6. Zmienne CSS (customizacja)

:root { --custom-primary: #8ebf29; /* Kolor główny (zielony) */ --custom-primary-rgb: 142, 191, 41; --label-color: #0d6efd; /* Kolor labeli (niebieski) */ --section-border-color: #8ebf29; /* Kolor kreski pod tytułem sekcji */ }
